# Benchmark Template

## 1. 背景
- 测什么？
- 为什么测？

## 2. 硬件环境
- CPU / GPU / NPU 型号
- 核心数 / 频率 / 内存
- 操作系统

## 3. 软件环境
- 编译器版本
- 数学库版本
- 驱动 / Runtime 版本

## 4. 测试配置
- 输入规模
- batch size
- 数据类型（fp32 / fp16 / int8 等）
- 线程数 / block size / tile size

## 5. 指标
- latency
- throughput
- GFLOPS / TFLOPS
- bandwidth
- cache miss
- occupancy

## 6. 结果
- 表格 / 图
- 最佳配置
- 异常点

## 7. 结论
- 瓶颈在哪里？
- 哪些优化有效？
- 下一步怎么做？
